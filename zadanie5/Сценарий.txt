1.Получение всех задач
Нужно выполнить GET запрос /products
Ожидаемый результат: Статус код 200. В ответе возвращается список (пустой [] или содержащий посты).

2.Получение одной задачи по id
Нужно выполнить GET запрос по /products/one/{item_id}
item_id является уникальным идентификатором задачи, чтобы его получить, нужно выполнить запрос /tasks и оттуда взять id
Ожидаемый результат:
Если задача есть: Статус код 200. Возвращается объект с полной информацией о продукте (поля: id, title, description, reviews, rating).
Если задачи нет: Статус 404 Not Found.

3.Создание и добавление задачи в список
Нужно выполнить POST запрос /products/create и передать в него 1 параметра title (строка) description (строка) reviews (список строк) rating (число от 1 до 5)
Параметр complete изначально False
Ожидаемый результат: Создастся продукт и вернётся сообщение "Товар создан".
Проверка: запросить пост по GET запросу /products/one/{item_id}
item_id является уникальным идентификатором задачи, чтобы его получить, нужно выполнить запрос /products и оттуда взять id

4.Получение несуществующего продукта по ID
Нужно выполнить GET запрос по /products/one/несуществующий_id
Ожидаемый результат:
Статус код 200 (текущая реализация). В ответе отсутствует объект продукта или возвращается пустой результат.

5.Редактирование задачи по id
Нужно выполнить PUT запрос по /products/update/{item_id} и передать поля, которые нужно изменить title(строка) description(строка) reviews(список строк) rating(число ≤ 5)
Ожидаемый результат: обновится передаваемое поле у товара
Проверка: запросить задачу по GET запросу /products/one/{item_id} после обновления
item_id является уникальным идентификатором задачи, чтобы его получить, нужно выполнить запрос /products и оттуда взять id

6. Обновление продукта с недопустимым рейтингом
Действия:
Нужно выполнить PUT запрос /products/update/{item_id} с параметром rating=6
Ожидаемый результат:
Статус код 200. В теле ответа сообщение: "Оценка должна быть меньше или равна 5". Данные продукта не изменятся.
Проверка:Запросить продукт через GET /products/one/{item_id} – рейтинг останется прежним.

7.Удалить продукт из списка
Нужно выполнить DELETE запрос по /products/delete/{item_id}
Ожидаемый результат: задача удалится из списка и будет сообщение "Продукт удален"
Проверка: запросить все посты по GET запросу /products
