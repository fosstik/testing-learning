1.Получение всех постов
Нужно выполнить GET запрос /posts
Ожидаемый результат: Статус код 200. В ответе возвращается список (пустой [] или содержащий посты).

2.Получение однго поста по id
Нужно выполнить GET запрос по /posts/one/{item_id}
Ожидаемый результат:
Если пост есть: Статус код 200, возвращается объект с постом.
Если поста нет: Статус 404 Not Found.

3.Создание и добавление поста в список
Нужно выполнить POST запрос /posts/create и передать в него 3 параметра title:str, decription:str, comments[str]
Ожидаемый результат: создастся задача и высветится надпись "Запись создана".
Проверка: запросить пост по GET запросу /posts/one/{item_id}

4.Редактирование поста по id
Нужно выполнить PUT запрос по /posts/update/{item_id} и передать поля, которые нужно изменить title:str, decription:str, comments[str]
Ожидаемый результат: обновится передаваемое поле у поста
Проверка: запросить задачу по GET запросу /posts/one/{item_id} после обновления

5.Удалить пост из списка
Нужно выполнить DELETE запрос по /posts/delete/{item_id}
Ожидаемый результат: задача удалится из списка и будет сообщение "Post deleted"
Проверка: запросить все посты по GET запросу /posts
