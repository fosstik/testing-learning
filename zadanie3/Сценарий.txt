1.Получение всех задач
Нужно выполнить GET запрос /tasks
Ожидаемый результат: Статус код 200. В ответе возвращается список (пустой [] или содержащий посты).

2.Получение одной задачи по id
Нужно выполнить GET запрос по /task/one/{item_id}
item_id является уникальным идентификатором задачи, чтобы его получить, нужно выполнить запрос /tasks и оттуда взять id
Ожидаемый результат:
Если задача есть: Статус код 200, возвращается объект с задачей.
Если задачи нет: Статус 404 Not Found.

3.Создание и добавление задачи в список
Нужно выполнить POST запрос /task/create и передать в него 1 параметра title:str
Параметр complete изначально False
Ожидаемый результат: создастся задача и высветится надпись "Задача создана".
Проверка: запросить пост по GET запросу /task/one/{item_id}
item_id является уникальным идентификатором задачи, чтобы его получить, нужно выполнить запрос /tasks и оттуда взять id

4.Редактирование задачи по id
Нужно выполнить PUT запрос по /task/update/{item_id} и передать поля, которые нужно изменить title:str, complete:bool
Ожидаемый результат: обновится передаваемое поле у задачи
Проверка: запросить задачу по GET запросу /task/one/{item_id} после обновления
item_id является уникальным идентификатором задачи, чтобы его получить, нужно выполнить запрос /tasks и оттуда взять id

5.Удалить пост из списка
Нужно выполнить DELETE запрос по /task/delete/{item_id}
Ожидаемый результат: задача удалится из списка и будет сообщение "Задача удалена"
Проверка: запросить все посты по GET запросу /tasks
