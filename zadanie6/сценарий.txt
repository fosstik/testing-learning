1.Создание и добавление книги
Нужно выполнить POST запрос /books с параметрами title и author
Ожидаемый результат: Статус код 200. Возвращается сообщение "Книга добавлена". 
Проверка: Запросить все книги через GET /books - в списке должна быть добавленная книга.

2.Удаление книги
Нужно выполнить DELETE запрос /books/delete/{id} id берётся из списка при запросе /books
id является уникальным идентификатором задачи, чтобы его получить, нужно выполнить запрос /books и оттуда взять id
Ожидаемый результат: Статус код 200. Возвращается сообщение "Книга удалена". 
Проверка: Запросить все книги через GET /books - в списке не должно быть удалённой книги.'

3.Регистрация читателя
Нужно выполнить POST запрос /readers с параметром name
 Ожидаемый результат: Статус код 200. Возвращается сообщение "Читатель зарегистрирован". 
 Проверка: Читатель должен появиться в системе по запросу .

4.Получение всех читателей
Нужно выполнить GET запрос /readers 
Ожидаемый результат: Статус код 200. Возвращается список (пустой [] или содержащий читателей). 
Проверка: После регистрации читателей, запрос должен возвращать их список.

5.Выдача книги читателю
Нужно выполнить POST запрос /borrow с параметрами bookId и readerId
Ожидаемый результат: Статус код 200. Возвращается сообщение "Книга выдана".
Проверка: Статус книги должен измениться на недоступный.

6.Возврат книги
 Нужно выполнить POST запрос /return с параметром bookId, bookId берётся из списка при запросе /books 
 Ожидаемый результат: Статус код 200. Возвращается сообщение "Книга возвращена".
 Проверка: Статус книги должен измениться на доступный.

7.Выдача уже занятой книги
Действия: Попытаться выдать книгу двум разным читателям подряд
Ожидаемый результат: Статус код 400. 
Возвращается сообщение "Невозможно выдать книгу".

8. Возврат книги, которая не была взята 
Действия: Нужно выполнить POST запрос /return для доступной книги 
Ожидаемый результат: Статус код 400. Возвращается сообщение "Невозможно вернуть книгу
